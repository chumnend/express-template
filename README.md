## Project Setup

### Prerequisites
- Install Node.js
- Set environment variables

### Environment Variables
Copy and rename `env.example` to `.env`. Fill out the following variables.

```
# set for runtime environment
NODE_ENV=
PORT=

```

### Start
```
npm start
```
